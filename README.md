## Nombre del proyecto
Sistema de Cajero Automático

## Objetivo del sistema

Desarrollar un sistema en Python que simule el funcionamiento básico de un cajero automático, permitiendo al usuario realizar diferentes operaciones bancarias mediante un menú interactivo.

El proyecto aplica estructuras lógicas como condicionales y ciclos repetitivos, además de organizar el código de manera clara y estructurada.

## Descripción del sistema

El sistema permite al usuario ingresar una tarjeta, validar sus datos y acceder a un menú de opciones bancarias.

Las operaciones disponibles son:

- Consultar saldo
- Depositar dinero
- Retirar dinero
- Retiro sin tarjeta
- Transferir dinero
- Consultar historial de movimientos
- Cambiar contraseña
- Pagar servicios
- Recargar saldo de celular
- Bloquear tarjeta
- Salir del sistema


## Funcionamiento

1. El usuario ingresa la tarjeta.
2. El sistema solicita número de cuenta y contraseña.
3. Se validan los datos del usuario.
4. Si los datos son correctos, aparece el menú principal.
5. El usuario selecciona una operación.
6. El sistema procesa la solicitud y actualiza la información.
7. Se guarda el movimiento realizado.
8. El usuario decide si desea continuar o salir.

El proyecto utiliza:
